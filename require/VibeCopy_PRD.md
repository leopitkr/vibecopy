# 📘 셀러 전용 AI 카피 머신 PRD (Product Requirements Document)

작성일: 2026-02-13

---

# 1. 🎯 프로젝트 개요

## 1.1 제품명 (가칭)

**VibeCopy -- 셀러 전용 AI 판매 카피 패키지 생성기**

## 1.2 목표

- 3개월 내 유료 사용자 50\~100명 확보
- 월 매출 300\~500만원 달성
- 확장 구조 설계로 월 2,000만원 규모까지 확장 가능하도록 설계

## 1.3 타겟 고객

- 스마트스토어 / 쿠팡 파트너스 셀러
- 인스타그램/틱톡 공동구매 운영자
- 월 매출 300만\~3,000만원 규모 1인 셀러

---

# 2. 🔥 핵심 문제 정의

## 2.1 고객의 고통

- 상세페이지 문구 작성이 오래 걸림
- 숏폼 대본 제작이 번거로움
- 매번 카피를 새로 고민해야 함
- ChatGPT는 범용적이라 결과가 들쭉날쭉

## 2.2 우리가 해결하는 가치

> URL 또는 상품 정보 입력 → 즉시 판매 가능한 "카피 패키지" 생성

---

# 3. 🚀 MVP 기능 정의

## 3.1 핵심 기능: 원클릭 판매카피 패키지

### 입력

- 상품 URL 또는 상품명 + 특징 5줄
- 판매 채널 선택 (스마트스토어 / 쿠팡 / 공동구매 / 숏폼)
- 바이브 선택 (신뢰형 / 후기형 / 자극형 / 프리미엄형 / 공구특화)

### 출력 패키지

- 후킹 헤드라인 10개
- 핵심 베네핏 5개
- 공동구매 멘트 3종
- 숏폼 15초 대본 2종
- CTA 문구 5개
- 금칙어 리스크 체크

---

## 3.2 사용자 계정 기능

- 로그인/회원가입
- 생성 기록 저장
- 재생성 버튼
- 복사 버튼

---

## 3.3 요금제 구조

### Free

- 하루 3회 생성 제한

### Standard (19,000원)

- 월 500크레딧
- 전체 바이브 사용 가능
- 히스토리 저장

### Pro (49,000원)

- 무제한 생성
- 브랜드 보이스 저장
- CSV 다운로드
- A/B 카피 자동 변형
